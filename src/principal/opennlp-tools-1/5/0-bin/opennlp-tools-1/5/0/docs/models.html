Models

  
 

The models for each of the components within OpenNLP Tools are linked at the bottom of this page. These models are designed to work with the 1.4 version of OpenNLP Tools. The models are compressed and have a .gz extension. They need to remain compressed to be used with the OpenNLP tools package. If your browser wants to uncompresses these files for you, prevent this by downloading them with a right or alternate click and selecting the option to save the contents of the link instead of clicking on the link directly.

The models are organized by language, and then by type of processing. The names for each are fairly self-explanatory. For information concerning how to run the tools with these models consult the "Running the Tools" section of the README file in the distribution.

Having read the above, feel free to download the v1.4 models here.
